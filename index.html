🃏 마이티 룰

기본 구성

5인 게임 · 53장(52장+조커) · 각자 10장, 바닥 3장

카드 강약 (기루 있을 때)

조커 > 마이티(스A, 기루가 스페이드면 다A) > 기루패 > 선패 문양 > 나머지

비딩

13~20 사이로 공약 선언. 시작자는 패스 불가. 20 선언시 자동 종료.

바닥패

주공이 바닥 3장을 가져와 13장 중 3장을 버림. 공약 변경 가능 (확인 전 +1, 확인 후 +2)

프렌드

특정 카드를 가진 사람이 프렌드. 노프렌드 선언 가능(배율 2배)

딜미스

10,J,Q,K,A=+1점, 조커=-1점 합계가 0이하면 딜미스 가능. 10원 납부 후 재딜링.

승패

주공팀 획득 점수 ≥ 공약 → 주공 승리 / 미달 → 시민 승리

특수 규칙

런(20점): 배율 2배 / 백런(10점 미만): 배율 2배 / 노기루: 배율 2배 / 노프렌드: 배율 2배 (중복 적용)
